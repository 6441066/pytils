Для того, чтобы воспользоваться PyTils в Django, нужно:

  1. Добавить 'pytils' в INSTALLED_APPS
  2. В шаблоне загрузить соответствующий компонент PyTils (подробности см. в примерах к компонентам)
  3. Вставить в нужном месте искомый тег/фильтр.

Компоненты PyTils, доступные для загрузки:

  • pytils_dt - работа с датами
  • pytils_numeral - работа с числами
  • pytils_translit - простая транслитерация

Замечание: в случае какой-либо ошибки фильтры/теги возвращают значение 'unknown' (для Django это нестандарт)
